# Айрин | Локальный чат PWA

Мобильная PWA-оболочка в светлом iOS/Cupertino-стиле для резервного локального чата Айрин. Работает поверх существующего сервиса `airin-local-chat` на `http://127.0.0.1:8790`, но сама его не перезапускает и не меняет runtime.

## Возможности

- AJAX-проверка `/health`, `/api/status`, `/api/history`, `/api/queue`;
- отправка сообщений через `/api/chat`;
- кнопка **Позвать** для быстрого живого пинга Айрин;
- панель **Выгрузки**: история, статус, запрос сырой нити;
- управление режимом удержания через `/api/hotoff`;
- локальный вход оператора и мастер создания админов;
- PWA manifest и service worker для офлайн-shell;
- fallback: если AJAX не проходит, текст копируется для ручной отправки.
- отдельный локальный прокси `server.py`, чтобы PWA и AJAX были same-origin и не зависели от CORS.

## Ограничения

Локальный вход защищает интерфейс на устройстве, но не заменяет серверную авторизацию. FaceID / TouchID в этой версии отмечается как локальный быстрый вход; полноценная WebAuthn-проверка потребует backend.

## Запуск

Рекомендуемый запуск через локальный прокси из корня репозитория:

```bash
python local-chat/server.py --port 8792 --target http://127.0.0.1:8790
```

Открыть:

```text
http://127.0.0.1:8792/
```

Статический предпросмотр без AJAX:

```bash
python -m http.server 8789
```

```text
http://127.0.0.1:8789/local-chat/
```
